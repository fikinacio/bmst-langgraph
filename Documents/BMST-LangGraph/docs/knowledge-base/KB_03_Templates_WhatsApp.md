# Knowledge Base — Templates de Mensagens WhatsApp

## Princípios de Comunicação em Angola

1. **Tom:** Próximo mas profissional. Não formal a ponto de ser frio. Não informal a ponto de perder credibilidade.
2. **Língua:** Português angolano. Sem gírias excessivas. Sem anglicismos técnicos.
3. **Tamanho:** Mensagens curtas. Máx. 3 parágrafos por mensagem. Usar emojis com moderação (1-2 por mensagem).
4. **Foco:** Sempre orientado ao benefício do cliente, não às features técnicas.
5. **Nunca:** Mencionar "IA", "algoritmo", "machine learning" — falar em "assistente inteligente", "sistema automático", "solução digital".

## HUNTER — Templates de Prospecção

### Template 1: Primeiro Contacto (Saúde)
```
Bom dia [Nome],

Vi que a [Clínica/Hospital] tem uma presença muito activa em Luanda. Trabalho com clínicas privadas a implementar assistentes automáticos no WhatsApp — os pacientes marcam consultas, recebem lembretes e tiram dúvidas sem precisar de ligar ou aguardar.

Teria 10 minutos esta semana para conversarmos?
```

### Template 2: Primeiro Contacto (Hotelaria/Restauração)
```
Boa tarde [Nome],

Trabalho com hotéis e restaurantes em Luanda a automatizar o atendimento ao cliente no WhatsApp. Os clientes reservam, consultam preços e recebem confirmações sem intervenção da equipa — que fica livre para focar no serviço presencial.

Posso mostrar-lhe como funciona numa conversa rápida?
```

### Template 3: Primeiro Contacto (Retalho/Distribuição)
```
Bom dia [Nome],

Tenho acompanhado o crescimento do [Nome da Empresa] e gostaria de apresentar uma solução que temos implementado em distribuidores angolanos: atendimento automático 24h no WhatsApp para encomendas, consulta de stock e suporte a clientes.

Tem disponibilidade para uma conversa esta semana?
```

### Template 4: Follow-up Dia +3 (sem resposta)
```
Boa tarde [Nome],

Deixei uma mensagem há alguns dias sobre automação do atendimento da [Empresa] no WhatsApp. Percebi que provavelmente tem a agenda cheia — é completamente normal.

Posso enviar um exemplo de como funciona para ver se faz sentido para o vosso negócio?
```

### Template 5: Follow-up Dia +7 (último toque)
```
Bom dia [Nome],

Última mensagem da minha parte — não quero ser inconveniente.

Se tiver interesse em reduzir o volume de chamadas da equipa e melhorar a resposta aos clientes fora do horário de atendimento, estarei disponível. Caso contrário, boa continuação.
```

## CLOSER — Templates de Qualificação e Proposta

### Template 6: Mensagem de Diagnóstico
```
Obrigado pelo interesse, [Nome]! Para preparar a melhor solução para a [Empresa], precisava de perceber melhor a vossa situação actual.

Posso fazer-lhe 3 perguntas rápidas?

1️⃣ Quantas mensagens/chamadas de clientes recebem por dia em média?
2️⃣ Qual é o principal motivo de contacto (marcações, preços, informações, reclamações)?
3️⃣ Têm alguém dedicado ao atendimento ao cliente?
```

### Template 7: Envio de Proposta
```
[Nome], segue em anexo a proposta que preparei para a [Empresa].

Estruturei a solução com base no que partilhou: [RESUMO_1_LINHA_DO_PROBLEMA].

Pontos principais da proposta:
• O que vamos construir
• Prazo de entrega: [X] semanas
• Investimento total: [VALOR] AOA

Estou disponível para esclarecer qualquer dúvida. Prefere falar por aqui ou marcar uma chamada?
```

### Template 8: Follow-up Proposta Dia +3
```
Bom dia [Nome],

Gostaria de saber se teve oportunidade de analisar a proposta que enviei.

Existe algum ponto que queira esclarecer ou ajustar? Estou aqui para isso.
```

### Template 9: Follow-up Proposta Dia +7
```
[Nome], bom dia.

Sei que as decisões levam o seu tempo — completamente compreensível.

Se a proposta estiver dentro do que procuram, posso reservar disponibilidade para iniciar o projecto na próxima semana. Caso precise de ajustes no âmbito ou no valor, podemos conversar.
```
