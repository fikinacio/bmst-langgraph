## Posicionamento
> "Não vendemos tecnologia — desenhamos a vossa arquitectura operacional."

A BMST é uma empresa de tecnologia angolana especializada em IA aplicada, automação de processos e desenvolvimento digital. O nosso diferencial não é a tecnologia em si — é a capacidade de identificar onde a tecnologia resolve problemas reais de negócio e implementá-la de forma que funciona no contexto angolano.

## Serviços (por ordem de prioridade comercial)

### 1. Websites WordPress + Cursor (porta de entrada)
Websites institucionais e comerciais modernos, rápidos e mobile-first. Desenvolvidos com WordPress (CMS) e Cursor (AI-assisted coding). Inclui SEO básico, formulário de contacto, integração WhatsApp.

### 2. Chatbots e Assistentes WhatsApp com IA
Assistentes virtuais inteligentes no WhatsApp Business. Atendem clientes 24/7, respondem perguntas frequentes, qualificam leads, fazem marcações, e encaminham para humano quando necessário. Tecnologia: n8n + Dify + Evolution API + Claude API.

### 3. Automação de Processos Operacionais
Eliminação de tarefas manuais e repetitivas via automação. Exemplos: onboarding de clientes, gestão de stocks, relatórios automáticos, follow-up de vendas, integração entre sistemas. Tecnologia: n8n + Dify + APIs específicas.

### 4. Análise de Dados com IA
Transformação de dados em insights accionáveis. Dashboards automáticos, relatórios inteligentes, detecção de padrões, previsão de vendas. Para empresas com dados mas sem capacidade de os interpretar.
